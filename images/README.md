# Photos go here

Drop the shop's photos in this folder, then reference the filename in
`js/data.js` (for products) or swap the placeholder `<div class="ph">` blocks
in the HTML pages (for hero/about/services photos).

Suggested shot list to request from the owner:
- [ ] Storefront exterior (about page + home page)
- [ ] Her at the work table arranging flowers (about page)
- [ ] 1 photo per arrangement being sold (shop page) — landscape, good light
- [ ] A wedding she's done (services page)
- [ ] A delivery van / delivery in progress if she has one (services page)
- [ ] Inside of the shop / cooler full of flowers (anywhere!)

Keep files under ~500 KB each — resize big phone photos at https://squoosh.app
